# Issue #25

Source: https://github.com/konard/anime-avatar/issues/25

Title: international phonetic alphabet accurate mouth talking animation

Labels: documentation, enhancement

## Body

We should be able to make animation of talking as true to real physics as possible, based on IPA (international phonetic alphabet), which encodes actual positions or 3D shapes of mouth and tongue.

If done perfectly it also may allow to do generation of 3D sound by calculating air movement in relation to IPA guided mouth and tongue positions/forms/shapes.

So we need conversion from English to IPA, and once we got IPA we can convert it to actual mouth animation. That should be small text area with a switch in editor, by default turned off as experimental feature.

Find best possible solution for that in the internet and do it.

We need to collect data related about the issue to this repository, make sure we compile that data to `./docs/case-studies/issue-{id}` folder, and use it to do deep case study analysis (also make sure to search online for additional facts and data), list of each and all requirements from the issue, and propose possible solutions and solution plans for each requirement (we should also check known existing components/libraries, that solve similar problem or can help in solutions).
